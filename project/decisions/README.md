# Decisions

Use this directory only for frozen project-level decisions worth re-reading later.

Do not create a decision record for every task choice.
Create one when the decision changes the project's durable direction, boundary, or control model.

## Naming

Store each record as:

`DEC-YYYYMMDD-NNN-short-name.md`

## Typical content

- decision status
- context
- options considered
- chosen option
- reasoning
- revisit trigger
